A powerful, full-stack cross-platform **Expense Tracker** application designed to help users monitor their daily, monthly, and yearly budgets efficiently. Built using **Flutter** for a beautiful user experience and backed by a robust **ASP.NET Core Web API**.

---

## 🚀 Key Features

* 📅 **Multi-Timeline Tracking:** Monitor expenses effortlessly across **Daily**, **Monthly**, and **Yearly** views.
* 🏷️ **Smart Category Management:** Assign expenses to specific categories for better insights.
* ⚠️ **Budget Thresholds:** Set **monthly spending limits** for individual categories to avoid overspending.
* 📊 **Visual Analytics:** View detailed statistics and expense breakdowns through **interactive graphs**.
* 💾 **Local & Remote Storage:** Uses **Hive** for fast local caching and an online **REST API** for server data sync.

---

## 🛠️ Tech Stack

### 📱 Frontend
- **Framework:** Flutter (Dart)
- **Local Database:** Hive

### 🖥️ Backend & Database
- **API Framework:** ASP.NET Core Web API (C#)
- **Database Server:** Microsoft SQL Server
